## 功能预览（Feature Preview）

> 本文是对项目核心能力的图文预览，更适合快速了解「这个工具能帮我做什么」。

### 1. 核心自动化能力

- **每日签到自动化**
  - 自动完成网易云音乐「每日签到」
  - 自动完成「音乐人云豆签到」
  - 按 `SEND_TIME` 每天定时执行，无需人工干预

![每日任务示例](./images/每日任务.png)


- **音乐人动态分享自动化**
  - 自动从歌单/推荐中随机选择歌曲分享（降低风控）
  - 分享 10 秒后自动删除动态，不打扰好友
  - 支持按照间隔天数（如 3 天）控制分享频率

![自动发动态示例](./images/自动发动态.png)

### 2. 多账号与任务调度

- **多账号支持**
  - 优先通过 Redis 维护多个账号配置
  - Redis 不可用时可降级使用本地 `accounts.json`
  - 每个账号单独记录任务执行时间

- **任务调度与限流**
  - 每日任务固定时间执行
  - 间隔任务按 `EXECUTION_INTERVAL_DAYS` 检查是否需要执行
  - 每月最大发送次数 `MAX_MONTHLY_SENDS` 限制，防止过度操作

### 3. 登录与风控优化

- **智能登录策略**
  - Redis 可用时优先复用其中的 Cookie
  - Redis 不可用时复用 Playwright profile 登录态
  - 失效再自动登录，减少登录次数

- **Playwright 加持**
  - 提供网页端扫码/密码登录流程（Playwright）
  - 通过模拟真实网页行为，显著降低：
    - `301 用户未登陆`
    - 「安全验证失败」「分享异常」等风控问题

![登录示例](./images/登录.png)

### 4. VIP 权益与附加任务

- **VIP 自动领取**
  - 自动执行与音乐人 VIP 相关的任务
  - 自动领取可用的 VIP 权益（具体以网易云音乐为准）
  - 
![自动续期任务](./images/自动续期任务.png)

- **任务失败自动重试**
  - 网络波动 / 接口偶发失败时，自动重试多次
  - 提升任务整体成功率

### 5. 通知与可观测性

- **企业微信 Webhook 通知**
  - 支持配置 `WECOM_WEBHOOK_KEY`
  - 将任务执行结果、异常信息推送到企业微信群
  - 适合部署在服务器上无人值守运行
  - 
![企业微信通知](./images/企业微信通知.png)

- **日志与执行记录**
  - 详细区分定时调度日志与核心执行日志
  - Redis 可用时在 Redis 中记录执行时间，不可用时写入本地状态文件
  - 方便排查问题与观察任务命中情况

### 6. 部署体验

- **Docker 一键运行**
  - 提供 `Dockerfile` 与 `docker-compose.yml`
  - 支持日志目录挂载、时区配置、环境变量配置

- **本地 / 服务器通用**
  - 支持直接 `python main.py` 运行
  - 也可部署在云服务器上长期运行
